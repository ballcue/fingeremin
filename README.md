# Theremin MIDI
## Dependencies
* OpenCV 3
* JACK
  * jackd
  * libjack-jackd2-dev
* cmake and make for compiling

## Compiling
* Go to build
* Run _cmake ._
* Run _make main_

An exectuable will be written at bin/main.

## Execution
First make sure there's a process listening for MIDI messages at an open MIDI port.
* Run bin/main
* Follow on-screen instructions
